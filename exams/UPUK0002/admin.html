Adding / Editing Questions

  1. Go to your GitHub repo
  2. Open questions.json and click the pencil icon to edit
  3. Each question looks like this:
{
  "section": "Section A — Terminology",
  "q": 1,
  "text": "Your question here?",
  "marks": 1,
  "image": "images/my-photo.jpg",
  "answer": "The correct answer."
}

Set "image" to null if no image is needed.

Adding Images

  1. Go to the images/ folder in your GitHub repo
  2. Click Add file > Upload files
  3. Drag your image in (JPG, PNG, etc.)
  4. Commit the change
  5. In questions.json, set the image field to: "images/your-filename.jpg"

That's it — the image will show above the answer box for that question.

Creating a New Exam

  1. In your GitHub repo, click Add file > Create new file
  2. Type exams/UPUK0002/questions.json as the filename (this creates the folder automatically)
  3. Paste in a new set of questions (copy the format from the existing questions.json)
  4. Upload any images to exams/UPUK0002/images/
  5. Copy index.html into that folder too (or I can set that up for you)
  6. The new exam will be at: unipresuk.github.io/electrical-exam/exams/UPUK0002/

Multiple Choice Questions

Add an "options" array to any question:

{
  "q": 11,
  "text": "What colour is a neutral conductor?",
  "marks": 1,
  "image": null,
  "options": ["Brown", "Blue", "Green/Yellow", "Black"],
  "answer": "Blue"
}
